Welcome

This lightweight web app walks you through an adaptive interview. Scores and notes are stored locally on this device so you can pause and resume at any time.

  1. Review the rubric for each question.
  2. Capture a rating with 1, 2, or 3, or use the buttons.
  3. Record optional notes to support the final recommendation.
